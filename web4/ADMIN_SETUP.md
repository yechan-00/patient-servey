# 관리자 설정 가이드

## 관리자 이메일 설정

관리자 권한을 부여하려면 `web4/src/config/adminConfig.js` 파일을 열고 `ADMIN_EMAILS` 배열에 관리자 이메일을 추가하세요.

### 설정 방법

1. `web4/src/config/adminConfig.js` 파일을 엽니다.

2. `ADMIN_EMAILS` 배열에 관리자 이메일을 추가합니다:

```javascript
export const ADMIN_EMAILS = [
  "your-email@example.com", // 여기에 관리자 이메일을 추가
  // 추가 관리자 이메일을 더 추가할 수 있습니다
];
```

3. 파일을 저장합니다.

4. 해당 이메일로 로그인하면 자동으로 관리자 권한이 부여됩니다.

### 관리자 권한

관리자는 다음 기능을 사용할 수 있습니다:

- ✅ 모든 게시글 수정/삭제
- ✅ 모든 댓글 수정/삭제
- ✅ 신고된 게시글/댓글 처리 (구현 예정)
- ✅ 사용자 제재 (구현 예정)
- ✅ 커뮤니티 관리 (구현 예정)

### 주의사항

- 관리자 이메일은 대소문자를 구분하지 않습니다.
- 관리자 권한은 로그인 시 자동으로 확인됩니다.
- 관리자 권한은 `community_users` 컬렉션의 `role` 필드에 `"admin"`으로 저장됩니다.
