## 상담 기록 상세 페이지 개선

### 1. 환자 건강 상태 요약 카드 추가

**현재 문제점**: 상담 기록 페이지에서 환자의 현재 건강 상태를 확인하려면 환자 상세 페이지로 이동해야 함

**개선안**:

- 상담 기록 페이지 상단에 환자 건강 상태 요약 카드 추가
  - 최근 설문 결과 요약 (위험도, 주요 점수)
  - 최근 설문일
  - 주요 건강 이슈 (정신건강, 생활습관 등)
  - 시각적 인디케이터 (위험도 배지, 트렌드 화살표)

**구현 예시**:

```jsx
<PatientHealthSummaryCard>
  <HealthStatusBadge riskLevel={patient.riskLevel} />
  <RecentSurveyInfo>
    최근 설문: {lastSurveyDate}
    전체 점수: {overallScore}
  </RecentSurveyInfo>
  <KeyHealthIssues>
    {keyIssues.map((issue) => (
      <IssueTag>{issue}</IssueTag>
    ))}
  </KeyHealthIssues>
</PatientHealthSummaryCard>
```

### 2. 상담 타임라인 개선

**현재 문제점**: 상담 노트가 단순 리스트로만 표시됨

**개선안**:

- 시각적 타임라인으로 변경
  - 상담 요청 → 예약 확정 → 상담 진행 → 상담 완료 단계별 표시
  - 각 단계별 타임스탬프와 담당자 정보
  - 상담 노트를 타임라인에 통합하여 시간순 흐름 파악 용이

### 3. 상담 노트 기능 강화

**현재 문제점**: 상담 노트가 단순 텍스트만 저장됨

**개선안**:

- **노트 카테고리/태그 시스템**
  - 상담 주제 분류 (심리 지원, 생활 습관, 의료 정보 등)
  - 중요도 표시 (중요/일반/참고)
- **노트 템플릿**
  - SOAP 형식 (Subjective, Objective, Assessment, Plan)
  - 빠른 입력을 위한 템플릿 제공
- **노트 검색 및 필터링**
  - 날짜 범위, 카테고리, 키워드로 검색
- **노트 수정/삭제 기능**
  - 작성자만 수정 가능
  - 수정 이력 추적

### 4. 상담 일정 관리 개선

**현재 문제점**: 상담 일정이 단순 텍스트로만 저장됨

**개선안**:

- **캘린더 뷰 통합**
  - 상담 일정을 캘린더 형식으로 표시
  - 다른 상담 일정과의 충돌 확인
- **알림 기능**
  - 상담 전날/당일 알림 (이메일/앱 내 알림)
  - 상담 후 후속 조치 알림
- **상담 일정 변경 이력**
  - 일정 변경 내역 추적
  - 변경 사유 기록

### 5. 상담 결과 요약 및 후속 조치

**현재 문제점**: 상담 후 후속 조치가 노트에만 기록됨

**개선안**:

- **상담 결과 요약 섹션**
  - 주요 논의 사항 요약
  - 결정 사항
  - 후속 조치 항목 (체크리스트)
- **후속 조치 추적**
  - 각 후속 조치의 완료 여부 추적
  - 다음 상담 예정일 설정
- **상담 효과 측정**
  - 상담 전후 환자 상태 비교
  - 설문 점수 변화 추적

### 6. 환자 연락 기능 통합

**현재 문제점**: 환자 연락이 별도 시스템으로 분리됨

**개선안**:

- **빠른 연락 버튼**
  - 전화, 문자, 이메일 바로가기
  - 연락 내역 기록 (통화 시간, 내용 요약)
- **연락 기록 타임라인**
  - 모든 연락 내역을 시간순으로 표시
  - 상담 노트와 연락 기록 통합

### 7. 문서 첨부 기능

**개선안**:

- **파일 첨부**
  - 상담 관련 문서, 이미지 첨부
  - 환자가 제공한 자료 보관
- **템플릿 문서 생성**
  - 상담 보고서 자동 생성
  - PDF 내보내기

## 대시보드 페이지 개선

### 1. 대시보드 레이아웃 개선

**현재 문제점**: 정보가 단순 리스트/테이블로만 표시됨

**개선안**:

- **위젯 기반 대시보드**
  - 사용자가 원하는 위젯을 드래그 앤 드롭으로 배치
  - 위젯 크기 조절 가능
  - 위젯 표시/숨김 설정
- **다중 뷰 모드**
  - 컴팩트 모드 (간단한 정보만)
  - 상세 모드 (모든 정보 표시)
  - 커스텀 모드 (사용자 정의)

### 2. 통계 및 분석 강화

**현재 문제점**: 기본적인 통계만 제공됨

**개선안**:

- **고급 통계 카드**
  - 월별/연도별 환자 증가 추이
  - 위험군 환자 비율 변화
  - 상담 완료율
  - 평균 상담 소요 시간
  - 환자 만족도 (설문 기반)
- **트렌드 분석**
  - 시계열 차트로 주요 지표 추이 표시
  - 예측 분석 (향후 위험군 환자 예상 수)
- **비교 분석**
  - 기간별 비교 (이번 달 vs 지난 달)
  - 카테고리별 비교 (암 종류별, 연령대별)
- **인사이트 카드**
  - AI 기반 인사이트 (예: "이번 주 위험군 환자 3명 증가")
  - 권장 사항 제시

### 3. 환자 목록 필터링 및 정렬 강화

**현재 문제점**: 기본적인 필터만 제공됨

**개선안**:

- **고급 필터**
  - 복합 필터 (여러 조건 동시 적용)
  - 저장된 필터 프리셋
  - 필터 히스토리
- **스마트 필터**
  - "오늘 상담 예정인 환자"
  - "일주일 이상 연락 없는 환자"
  - "최근 위험도가 상승한 환자"
  - "상담 후속 조치가 필요한 환자"
- **정렬 옵션 확대**
  - 다중 정렬 (예: 위험도 → 최근 상담일)
  - 사용자 정의 정렬

### 4. 작업 관리 및 알림 시스템

**개선안**:

- **할 일 목록 (To-Do List)**
  - 상담 예정, 후속 조치, 환자 연락 등 작업 관리
  - 우선순위 설정
  - 마감일 설정 및 알림
- **알림 센터**
  - 실시간 알림 (새 상담 요청, 위험군 환자 등)
  - 알림 카테고리별 필터링
  - 알림 읽음/안 읽음 관리
- **작업 할당**
  - 팀원 간 작업 할당
  - 작업 상태 추적

### 5. 캘린더 통합

**개선안**:

- **통합 캘린더 뷰**
  - 모든 상담 일정을 한눈에 확인
  - 월별/주별/일별 뷰 전환
  - 드래그 앤 드롭으로 일정 변경
- **캘린더 필터**
  - 담당자별, 환자별, 상태별 필터링
- **일정 충돌 감지**
  - 중복 일정 자동 감지 및 경고

### 6. 검색 기능 강화

**개선안**:

- **통합 검색**
  - 환자 이름, 전화번호, 상담 내용, 노트 등 모든 데이터 검색
  - 자동완성 및 검색 제안
- **고급 검색**
  - 불린 연산자 지원 (AND, OR, NOT)
  - 날짜 범위, 숫자 범위 검색
  - 정규식 검색
- **검색 결과 하이라이트**
  - 검색어 하이라이트
  - 검색 결과 요약

### 7. 보고서 및 내보내기

**개선안**:

- **자동 보고서 생성**
  - 일일/주간/월간 보고서 자동 생성
  - 주요 지표 요약
  - 트렌드 분석 포함
- **커스텀 보고서**
  - 사용자가 원하는 데이터만 선택하여 보고서 생성
  - 다양한 형식 지원 (PDF, Excel, CSV)
- **일괄 내보내기**
  - 선택한 환자 데이터 일괄 내보내기
  - Excel/CSV 형식 지원

### 8. 협업 기능

**개선안**:

- **팀 협업**
  - 팀원 간 메시지 교환
  - 환자 정보 공유
  - 작업 할당 및 추적
- **댓글 시스템**
  - 환자 정보, 상담 기록에 댓글 추가
  - @멘션 기능
- **활동 피드**
  - 팀원의 최근 활동 내역 표시
  - 환자 정보 변경 이력

## 병원 시스템 비교 기반 추가 기능

### 1. 환자 관리 시스템 (EMR/EHR 스타일)

**참고 시스템**: 전자의무기록(EMR), 병원정보시스템(HIS)

**추가 기능**:

- **환자 차트 뷰**
  - 환자 정보를 차트 형식으로 시각화
  - 시간순 정보 표시 (타임라인)
  - 주요 이벤트 마커 (진단, 수술, 상담 등)
- **의료 기록 통합**
  - 병원 EMR과 연동 (API 연동)
  - 진단서, 처방전 등 의료 문서 통합
- **알레르기 및 약물 정보**
  - 알레르기 정보 표시
  - 복용 중인 약물 정보
  - 약물 상호작용 경고

### 2. 케이스 관리 시스템

**참고 시스템**: 사회복지 케이스 관리 시스템

**추가 기능**:

- **케이스 파일 관리**
  - 환자별 케이스 파일 생성 및 관리
  - 케이스 단계별 진행 상황 추적
  - 케이스 목표 및 계획 수립
- **서비스 계획 (Service Plan)**
  - 개별 환자별 서비스 계획 수립
  - 목표 설정 및 달성도 추적
  - 정기 평가 및 계획 수정
- **리소스 연결**
  - 환자에게 필요한 자원 정보 제공
  - 외부 기관 연계 기록
  - 추천 서비스 목록

### 3. 평가 및 측정 도구

**참고 시스템**: 정신건강 평가 도구, 삶의 질 측정 도구

**추가 기능**:

- **정기 평가 스케줄링**
  - 자동 평가 일정 설정
  - 평가 미완료 알림
  - 평가 결과 비교 분석
- **다양한 평가 도구 통합**
  - PHQ-9 (우울증 평가)
  - GAD-7 (불안 평가)
  - 삶의 질 평가 (QOL)
  - 통증 평가 (NRS)
- **평가 결과 시각화**
  - 평가 점수 변화 그래프
  - 기준선 대비 비교
  - 개선/악화 트렌드 표시

### 4. 알림 및 리마인더 시스템

**참고 시스템**: 병원 스케줄링 시스템, CRM 시스템

**추가 기능**:

- **스마트 알림**
  - 위험군 환자 자동 감지 및 알림
  - 장기간 연락 없는 환자 알림
  - 상담 후 후속 조치 리마인더
- **알림 규칙 설정**
  - 사용자 정의 알림 규칙
  - 알림 채널 선택 (이메일, SMS, 앱 푸시)
  - 알림 빈도 조절
- **일정 리마인더**
  - 상담 전 자동 리마인더
  - 환자 생일, 기념일 알림
  - 정기 설문 리마인더

### 5. 데이터 분석 및 인사이트

**참고 시스템**: 병원 BI 시스템, 헬스케어 분석 플랫폼

**추가 기능**:

- **예측 분석**
  - 위험군 환자 예측 모델
  - 상담 필요도 예측
  - 자원 필요량 예측
- **코호트 분석**
  - 특정 그룹의 장기 추적
  - 그룹별 결과 비교
- **벤치마킹**
  - 다른 기관/팀과의 성과 비교
  - 업계 평균 대비 비교

### 6. 보안 및 규정 준수

**참고 시스템**: HIPAA, GDPR 준수 시스템

**추가 기능**:

- **접근 로그**
  - 모든 데이터 접근 기록
  - 사용자별 활동 로그
- **권한 관리 강화**
  - 역할 기반 접근 제어 (RBAC)
  - 세밀한 권한 설정
- **데이터 암호화**
  - 전송 중 및 저장 중 암호화
  - 민감 정보 마스킹
- **감사 추적**
  - 모든 데이터 변경 이력 추적
  - 변경자, 변경 시간, 변경 내용 기록

### 7. 모바일 최적화

**참고 시스템**: 모바일 헬스케어 앱

**추가 기능**:

- **반응형 디자인 강화**
  - 모바일에서도 모든 기능 사용 가능
  - 터치 친화적 UI
- **오프라인 모드**
  - 인터넷 연결 없이도 기본 기능 사용
  - 동기화 기능
- **모바일 전용 기능**
  - QR 코드로 환자 정보 빠른 조회
  - 음성 입력으로 노트 작성
  - 위치 기반 기능 (병원 내 환자 찾기)

### 8. 통합 및 연동

**참고 시스템**: 병원 정보 시스템 통합

**추가 기능**:

- **API 연동**
  - 병원 EMR 시스템과 연동
  - 외부 서비스 연동 (캘린더, 이메일 등)
- **데이터 동기화**
  - 실시간 데이터 동기화
  - 배치 동기화 옵션
- **웹훅 지원**
  - 외부 시스템으로 이벤트 전송
  - 자동화 워크플로우 구축

## 우선순위 추천

### 높은 우선순위 (즉시 구현 권장)

1. ✅ 환자 건강 상태 요약 카드 (상담 기록 페이지)
